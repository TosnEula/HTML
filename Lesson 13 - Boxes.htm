Lesson 13 - Boxes

Boxes

Width & Height

Using width and height can control the size of the box. Using percentages and ems will change the box according to the size of the screen or the box that it is in.

Limiting Width

the min-width property specifies the smallest size a box can be displayed at when the browser window is narrow, and the max-width property indicates the maximum width a box can stretch to when the browser window is wide.

Limiting Height

the min-height property specifies the smallest size a box can be displayed, and the max-height property indicates the maximum width a box can stretch to.

Overflow

Using the property overflow allows for the browser to adjust the content according to the specification.

  • hidden - Hides the information that has overflown.
  • scroll - Gives the content a scroll bar to view the extra content.

Parts of Boxes

  • Border - Every box has a border (even if it is not visible or is specified to be 0 pixels wide). The border separates the edge of one box from another.
  • Margin - Margins sit outside the edge of the border. You can set the width of a margin to create a gap between the borders of two adjacent boxes.
  • Padding - Padding is the space between the border of a box and any content contained within it. Adding padding can increase the readability of its contents.

Border Width

The property border-width to control the size of the border of a box. Can use pixels to specify size or use the preset:

  • thin
  • medium
  • thick

It is possible to specify the side of the border by using the properties, border-bottom-width, border-top-width, border-left-width, border-right-width.
