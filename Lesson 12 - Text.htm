Ways to Implement Fonts

Font-Family Property

The property font-family allows you to specify the typeface that should be used for any text inside the element(s) to which a CSS rule applies.
The value of this property is the name of the typeface you want to use.

  • You can specify a list of fonts separated by commas so that, if the user does not have your first choice of typeface installed, the browser can try to use an alternative font from the list.
  • Use double quotes for typefaces that are more than one word.

Font Size

The property font-size changes the size of the font in three different ways:

  • Percentages
  • pixels
  • EMs -An em is equivalent to the width of a letter m.

Downloaded Font Face

Using @font-face will download a font from a specified location should it not be normally on a computer.
Use src to specify the location.

Text Format

Bold

The property font-weight is used to make text bold. It has two settings:

  • bold
  • normal

Italic

The property font-style is used to make text italic. It has three settings:

  • italic
  • normal
  • oblique - Slanting the letters literally.

Uppercase and Lowercase

The property text-transform is used to change the case of words. It has three settings:

  • uppercase
  • lowercase
  • capitalize - the first letter of each word is capitalized.

Underline and Strike

The property text-decoration is used to add lines to the text. The settings are:

  • none
  • underline
  • overline
  • line-through
  • blink

Alignment

The property text-align is used to organize text. The settings are:

  • left
  • right
  • center
  • justify - This indicates that every line in a paragraph, except the last line, should be set to take up the full width of the containing box.

Vertical Alignment

The property verical-align is used to organize text according to a picture. The settings are:

  • baseline
  • sub
  • super
  • top
  • text-top
  • middle
  • bottom
  • text-bottom
  • Also uses numbers.

Indenting Text

The property text-indent indents lines of text. this uses pixels and ems to move text.

This is an example.

Drop Shadow

The property text-shadow adds a drop shadow to the text. this uses three values:

  • The first length indicates how far to the left or right the shadow should fall.
  • The second value indicates the distance to the top or bottom that the shadow should fall.
  • The third value is optional and specifies the amount of blur that should be applied to the drop shadow.
  • The fourth value is the color of the drop shadow.
