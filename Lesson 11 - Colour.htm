Colour

The color property affects the colour of the text. This can be done in three ways:

  • RGB Values
  • Hex Codes
  • colour names

Background Colour

The property background-color is used to change the background colour of individual elements.

Using the opacity property can change the brightness of the colour being applied. This value ranges from 1-0.NOTE: The fourth number of the RGB refers to the alpha.

HSL Colours

HSL meaning, Hue, Saturation, Lightness. the HSL property changes the three values name respectively.

  • Hue is measure on a range of 0-360
  • Saturation is represented in percentage
  • Lightness is represented as a percentage with 0% being white, 50% being normal and 100% being black
