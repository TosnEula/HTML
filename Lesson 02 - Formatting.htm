Lesson 02 - Formatting

Paragraphs

Simplying typing a paragraph normally and using the start-end elements create a paragraph.

Starting another paragraph is as simple as starting yet another p element.

Helpful Functions

Using the element b creates a BOLD marker.
Similarly the element i creates an itlaized marker.

sub and sup markers make subscripts and superscripts respectively. e=mc2 or CO2


hr / creates a horizontal line as such.

Semantic Markups

Semantic markups are elements that are not intended to affect the structure of the web page but add extra information.

em and strong appears as italics and bold respectively. Strong will show that the item has strong importance while em shows that theres a subtly difference.

BEWARE: pickpockets operate in this area.

I think Emily was first

Quotations

There are two kinds of quotes, the element blockquote and q

The blockquote is used for an entire paragraph and indents the sentences, while the quote is used for a sentence.

Did you ever stop to think, and forget to start again?

As A.A. Milne said, Some people talk to animals. Not many listen though. That's the problem.

Writing elements

abbr is the element used to show abbreviations for acornyms and titles. an attribute is used to display the definition.

Prof Stephen Hawking is a theoretical physicist and cosmologist.

NASA do some crazy space stuff.


cite does exactly what it implies.

A Brief History of Time by Stephen Hawking has sold over ten million copies worldwide.


dfn is great for when a word is defined and used. Chrome and firefox does not display the text any differently.

A black hole is a region of space from which nothing, not even light, can escape.


The address element specifically displays the address of an author.

homer@example.org

742 Evergreen Terrace, Springfield.


Changes to Content

del will show a word that has been deleted while ins shows words that was inserted.

It was the worst best idea she had ever had.


s indicates something that is no longer accurate but should not be deleted

Laptop computer:

Was $995

Now only $375
