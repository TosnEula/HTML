CSS Structure

Rules

A CSS rule contains two parts, a selector and declaration.

A selector indicates which element the rule applies to. Multiple rules can be indicated by using commas.

Declarations are used to tell how the element should be styled.

Declarations

Declarations are split into two parts, property and value

Properties indicate the aspects of the element you want to change.

Values specify the settings you want to use for the chosen properties.

Using External CSS

Using <link> applies the CSS stlye to the html webpage. It must be implemented in the head.

  • href - Given the vlaue of the directory of the CSS file.
  • type - This attribute specifies the type of document being linked to. The value should be "text/css".
  • rel - This specifies the relationship between the HTML page and the file it is linked to. The value should be stylesheet when linking to a CSS file.
