Adding Video to The Webpage

The <video> Element

The video element is used for adding videos to a webpage.

  • src - describes the path used to find the video.
  • poster - The thumbnail of the video before being played.
  • width & height - controls the size of the player.
  • controls - Chooses to show the playback of the video.
  • autoplay - decides if a video should play once the page loads.
  • loop - Makes a video play again once finished.
  • preload - determines what should be done to the video during the time it is not played.
    1. none - The browser should not load the video until the user pressesplay.
    2. auto - The browser should download the video when the page loads.
    3. metadata - The browser should just collect information such as the size, first frame, track list, and duration.

Using Source

The source element helps to add multiple video sources to a webpage.

  • src - describes the path used to find the video.
  • type - tell the browser what format the video is. Otherwise, it will download some of the video to see if it can play the file
  • codecs - The codec that was used to encode the video is supplied within the type attribute.Need to be memorised else cannot be known.
