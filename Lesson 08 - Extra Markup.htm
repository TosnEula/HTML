Extra Markup

Comments

using the element ! will add comments to the code.


Id Attribute

Also refered to as the Global Attribute it can be used on any element and can be used for individually modifing an element.

Class Attribute

The class attribute can be used to help distinguish elements that are the same.

This has no bearing on the page nor the html formatting, unless explictly told in CSS or similar coding languages.


Formatting

block elements are elements that will create a new line when added no matter the situation. Such as:

  • h1
  • p
  • ul
  • li

Inline elements are elements that will continue on the same line when created. Such as:

  • a
  • b
  • em
  • img

The div Element

div is used to seperate a block of code from another for easy identification. The id attribute can be used to help easily recognise the purpose or adding comments to the end helps with identification.


Span Element

span works similarly to div, although for inline text. Its purpose involves:

  • Contain a section of text where there is no other suitable element to differentiate it from its surrounding text.
  • Contain a number of inline elements.

Iframe Element

iframe is used to "cut" into the current webpage another. a common example is the use of map from google.

  • width
  • height
  • src
  • scrolling - Used to identify if the page should have a scroll since the dimensions don't cover the entire page, uses the value yes or no.(XHTML and HTML4 only)
  • frameborder - Gives the webpage a border using 1 or 0.(XHTML and HTML4 only)
  • seamless - can be used for pages that do not have scroll bars, does not require a value but usually given the value seamless.

Information About the Page

meta is an element that is put in the headign and used for multiple purposes. It is used for telling search engines about your page, who created it, and whether or not it is time sensitive.

  • name - Used to choose the value of the name of the attribute being set.
  • content - Used to give purpose to the attribute being named.
  • http-equiv - works similaraly to name, although mostly used for webpage loading inforamtion.
  • description - This contains a description of the page. This description is commonly used by search engines to understand what the page is about and should be a maximum of 155 characters. Sometimes it is also displayed in search engine results.
  • keywords - This contains a list of commaseparated words that a user might search on to find the page. In practice, this no longer has any noticeable effect on how search engines index your site.
  • robots - This indicates whether search engines should add this page to their search results or not. A value of noindex can be used if this page should not be added. A value of nofollow can be used if search engines should add this page in their results but not any pages that it links to.
  • author - gives the name of the author of the page
  • pragma - This prevents the browser from caching the page.
  • expires - Because browsers often cache the content of a page, the expires option can be used to indicate when the page should expire (and no longer be cached).
