Lesson 06 - Tables

Table Structure

Element table is used to start a table. Using the element tr to represent the start of a row in a table. Element td inputs data into the table, stands for table data.

Use th to represent the table head. A table should always have a heading even if left blank.

  • The attribute scope is needed to show if the heading is for the row or column. Represented by row or col

Formatting Tables

Using the attribute colspan on either the element tr or th will make the cell extend the designated amount.

Likewise, Using the attribute rowspan for the same elements vertically affects a cell.

Old Code

  • The attribute width used to be used on the element table to dicate how wide the table should be.
  • The attribute cellspacing used to be used on the element table to dicate how much space a cell would have.
  • The attribute cellpadding used to be used on the element table to dicate how much space there is between each cell.
  • The attribute border used to be used on the element table or td to dicate the thickness of the border between each cell.
  • The attribute bgcolor used to be used on the element table or td to change the background colour of a cell.
