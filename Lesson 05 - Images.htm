Lesson 05 - Images

Adding Images

Using the empty element img adds an image to the page, an empty element does not have a closing tag.

  • The attribute src gives the location of the image
  • The attribute alt gives the image a description. Used for visually impaired.
  • The attribute title shows the name of the image as a tool tip when hovered over by a mouse.
  • The attributes height and width adjusts the images size according to the amount of pixels.
  • The attribute align is used to organize the imagine. This is outdated and is only used on older sites.
  • The elements figure and figcaption allows for images to have captions.
